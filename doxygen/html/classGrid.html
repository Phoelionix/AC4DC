Detailed Description

The Grid class creates the coordinate grid on the interval from [r_min, r_max] with "num_grid_pts" points. The grid has not constant node spacing. It is constant for the following variable: S = R + beta*ln(R). According to Vladimir Dzuba the best value for beta=4.
